To-Do List (Immediate Actions & Priorities)

  1. **CRITICAL: Install Microsoft Visual C++ Build Tools**: This is essential for installing `qiskit-aer`.
    • Download from: Visual Studio Downloads
    • During installation, select the "**Desktop development with C++**" workload.
  2. **Install `qiskit-aer`**: After Build Tools are installed, run: 
    python -m pip install qiskit-aer
  3. **Run Helix Quantum Framework Script**: Once `qiskit-aer` is installed, navigate to `C:\Users\user\Documents` and run: 
    python -m HelixQuantumFramework.main
  4. **Debug, Fix, and Refactor Helix Quantum Framework**: Prioritize getting this "lego block" stable and functional.
  5. Install Git on PC: If not already done, ensure Git is installed and accessible.
  6. Start OpenSSH Server on PC: Manually start the `sshd` service via Windows Services.
  7. Transfer Termux Files: Run the `scp` command in Termux on your phone to transfer files to `C:\Users\user\Documents\QuantumBlockchain`. 
    scp -r <path_to_termux_files> <your_windows_username>@10.0.45.158:'/cygdrive/c/Users/<your_windows_username>/Documents/QuantumBlockchain/'
    Remember to replace <path_to_termux_files> and <your_windows_username>.
  8. Clone Quantum Blockchain Project: If the repository is not present (e.g., if `C:\Users\user\Documents\QuantumBlockchain` was deleted), clone it again: 
    "C:\Program Files\Git\cmd\git.exe" clone https://github.com/SnowVelda/Quantum-Blockchain-Project.git C:\Users\user\Documents\QuantumBlockchain
  9. Run Next.js Application:
    • Navigate to your project directory: cd C:\Users\user\Documents
    • Install dependencies: npm install
    • Start the development server: npm run dev
    • Access the main landing page at http://localhost:3000.
  10. Assess Storage Needs: Re-evaluate disk space after critical installations.

Once these immediate actions are completed, we can proceed with the next phases of the project as outlined in the roadmap.
